# Glory Desk screenshots

Upload PNG or JPG files here so they appear in the main [README.md](../../README.md) on GitHub.

## Recommended files

| File | What to capture |
|------|-----------------|
| `dashboard.png` | Main dashboard after login |
| `pos.png` | Point of Sale checkout screen |
| `inventory.png` | Product list / stock view |
| `reports.png` | Reports sidebar (e.g. Advanced Analytics) |
| `enterprise.png` | Enterprise hub (optional) |
| `login.png` | Login screen (optional) |

Use **lowercase filenames** exactly as above. PNG works best for UI captures.

After uploading, commit and push — GitHub will render them automatically in the README.
